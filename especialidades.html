HTML 5

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar e estruturar páginas na web. Ele define a organização dos elementos de uma página, como títulos, parágrafos, imagens, links e outros componentes. O HTML utiliza uma série de tags que informam ao navegador como exibir o conteúdo, sendo essencial para a construção de qualquer site ou aplicação web.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência e o layout de páginas web. Ele permite controlar cores, fontes, espaçamento, tamanhos, posicionamento de elementos e muito mais, separando o conteúdo (HTML) do design. Com o CSS, é possível criar layouts responsivos e visualmente atraentes, garantindo que as páginas sejam exibidas de maneira consistente em diferentes dispositivos e telas.
